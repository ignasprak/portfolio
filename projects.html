Explore Eire

Description

This website aims to bridge the gap that is missing in the Irish tourism industry, allowing users to search and visualise different attractions across Ireland, and to be able to simply plan a route through an itinerary. The goals of this website are to be as accessible as possible, be simple and visually enticing for any type of user.


Technologies

Next.js, Supabase, TailwindCSS, OpenMaps, Vercel
